SwiftOpenCV
==
OpenCV and Tesseract OCR from Swift and iOS

Benefits
-
- Example of using OpenCV/Tesseract from the Swift language
- Updated to use iOS 10.1 and XCode 8.1 as a base
- Build and run ready
- Swift 3.0

Getting Started
-
Download the XCode project and run the demo.  

Requirements
-
XCode 8.1   
iOS 10.1
